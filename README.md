# E-Commerce Database Project (MySQL)

## Project Description
This project demonstrates the design and implementation of an E-Commerce database using MySQL.
It includes customers, products, orders, and order items with proper constraints and relationships.

## Technologies Used
- MySQL
- SQL (DDL, DML, Constraints, Joins, Transactions)

## Database Structure
- customers
- products
- orders
- order_items

## Key Features
- Normalized database design
- Primary & Foreign key relationships
- Data integrity using constraints
- INNER JOIN & LEFT JOIN queries
- Transaction handling using COMMIT & ROLLBACK

## How to Run
1. Execute `schema.sql`
2. Execute `insert_data.sql`
3. Execute queries from `queries.sql`

## Sample Use Cases
- View customer order history
- Track product sales
- Identify customers with no orders
- Handle order transactions safely
